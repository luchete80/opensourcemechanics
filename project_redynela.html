ReDynELA

FEM Adaptive Remeshing Research Reference Code

ReDynELA is a reference FEM development line used for adaptive remeshing experiments and comparison against newer solver architectures. It remains relevant as a practical baseline for remeshing-oriented studies.

Current focus

  • Adaptive mesh remeshing experiments
  • Comparison against newer data-oriented solver implementations
  • Tracking distortion-control strategies under large plastic strain
