WeldFormGUI

GUI Pre/Post Python Export Workflow Tool

WeldFormGUI is a lightweight pre- and post-processing environment designed to reduce friction between model setup, solver export, execution, and result inspection.

Current focus

  • Geometry and mesh import
  • Boundary conditions and set management
  • Python-driven workflow scripting
  • Export pipelines for external solvers such as Abaqus and OpenRadioss
  • Integrated result visualization and energy plots
